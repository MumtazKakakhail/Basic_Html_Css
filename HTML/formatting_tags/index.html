Formatting tags

Formating tags are used to style and enhance the appearance of text on a webpage.
They define how text should be displayed such as bold, italic, underlined, or highlighted
providing better visual structure and emphasis.


1.b-tag:: Makes text bold.
Example: this is b tag
2. i-tag: Italicizes text.
Example: this is italic
3.U-tag: Used to underline text.
Example: This text is underlined
4.Small-tag: Makes text smaller than the surrounding texts.
Example: This is small
5.Big-tag: Makes text bigger than the surrounding texts.
Example: This is big
6.centre-tag: It is use to align content horizontally in the centre of the page or container.
Example: This text is centred

6. Subscript : Display subscript words.
Example:

H2O

7. Superscript: Display superscript words.
Example:

32=9
