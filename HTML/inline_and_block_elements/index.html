Inline Elements

  • Do not start on new line.
  • Only take as much width as the content inside them.

    • Example:

    This is a inline element.

    Inline Elements(Most commonly used):

    1. a: Anchor or Hyperlink
    2. Strong: Strong emphasis
    3. em: Emphasized text
    4. br: Line Break
    5. b: Bold text
    6. i: Italic text
    7. U: Underlined text
    8. Small: Smaller text
    9. Sub: Subscript
    10. Sup: Superscript
    11. Mark: Marked or Highlighted text

    Block Elements

  • Start on a new line and take up the full width of their container by defult.
  • Start on new line

    • Example:

    This is a block element.

    Block Elements(Most commonly used):

    • div:A generic container for flow content
    • P:Paragraph
    • h1-h6:Headings
    • ul:Unordered list
    • ol:Ordered list
    • li:List item
    • Pre:Preformatted list
    • dl:Description list
    • dt:Tern in a description list
    • dd:Description in a description list
