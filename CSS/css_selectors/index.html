CSS Selector

CSS selectors are patterns used to select the HTML elements you want to style.
They allow you to target specific elements on a webpage and apply styles to them.
Understanding CSS selectors is fundamental for web development, as they help you control the presentation of your webpage.

Universal Selector (*)*

The universal selector targets all elements on the page.
Example: css * { color: red; } This will make the text color of all elements red.

Class Selector (.)

- The class selector targets elements that have a specific class attribute. You use a dot (.) followed by the class name.
- Example: css .example { background-color: yellow; }

ID Selector (#)

- The ID selector targets a specific element with a particular ID.
You use a hash (#) followed by the ID name.
