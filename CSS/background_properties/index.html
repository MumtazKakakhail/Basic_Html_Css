Background-color

Sets the background color of an image.

Background-image

Sets an image as an background of an element.

Background Size

Defines the size of an background image.

Background Position

Specifies the position of the background image.

Background repeat

Controls weather the background image is repeated.

Background attachment

Specifies weather the background image scrolls with content.
