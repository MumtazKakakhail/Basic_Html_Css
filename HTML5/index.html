What is HTML

HTML or Hyper Text Markup Language, is the standard language used to create and structure content on the web.
It provides the basic framework for webpages by using tags to define elements such as texts, images, links, and multimedia.
Key features of HTML

1. Structures: HTML organizes content into headings, paragraphs, lists, and sections.
2. Multimedia support: It allows embedding of images, videos, and audios.
3. Hyperlinks: HTML enables linking between webpages using tags.

Founder of HTML

HTML was initially developed by Tim Berners-Lee in 1991. he is widely regarded as the inventor of World Wide Web(WWW).


HTML5:

HTML5 is the fifth and most recent version of HTML, which is standard language used to

Key Features of HTML5

HTML5 introduced sementic elements like <header> , <nav> , <section> , <article> , <aside> and <footer>

HTML5 Semantic Elements

Semantic elements clearly describe their meaning to both the browser and the developer. Here are the main ones:

  • <header> - Container for introductory content
  • <nav> - Navigation links
  • <main> - Main content of the document
  • <article> - Independent, self-contained content
  • <section> - Thematic grouping of content
  • <aside> - Content aside from the content
  • <footer> - Footer for a document or section

Block vs Inline Elements

Block Elements

Block elements always start on a new line and take up the full width available. Examples:

  • <div> - Generic block container
  • <p> - Paragraph
  • <h1> to <h6> - Headings
  • <ul> - Unordered list
  • <ol> - Ordered list

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples:

  • <span> - Generic inline container
  • <a> - Hyperlink
  • <img> - Image
  • <strong> - Strong importance
  • <em> - Emphasized text
